# Contributing

See the [Humanizer contributing guide on GitHub](https://github.com/Humanizr/Humanizer/blob/main/.github/CONTRIBUTING.md) for detailed instructions on building, testing, and submitting pull requests. The high-level workflow is:

1. Fork or clone the repository.
2. Install the required .NET SDK listed in `global.json`.
3. Run `dotnet test` for the `net10.0` and `net8.0` target frameworks.
4. Open a pull request that describes the change and links to any relevant issues.

Thank you for helping improve Humanizer!
