# API reference

Use the version selector in the sidebar to browse namespaces, classes, and members generated from the latest Humanizer builds.

- **Latest release** &mdash; the most recent `rel/vN.N` branch.
- **Latest dev (main)** &mdash; the current `main` branch, useful for previewing unreleased APIs.

The navigation tree is grouped by namespace. Select any type on the left to view its members, remarks, and examples.
